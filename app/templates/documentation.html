POST https://pentagonal-noiseless-evergreen.glitch.me/create_user

Crear un nuevo usuario.

Parámetro Descripción
nombre Nombre del usuario
email Email del usuario
contraseña Contraseña del usuario
apellido_paterno Apellido paterno del usuario
apellido_materno Apellido materno del usuario
ciudad Ciudad del usuario
estado Estado del usuario
telefono Teléfono del usuario

Response:

        Usuario registrado exitosamente!

GET https://pentagonal-noiseless-evergreen.glitch.me/win_or_loose

Get regresa un estado ya sea de Win o Lose para verificar si el usuario gana o pierde en el juego de la garra.

Parameter Description
No parameteros Este Endpoint no requiere ningun parametro

Response:

        Case 1: win
        Case 2: lose
